# MICROSCOPE SPRING BASIC

## Required 
 - maven3
 - java jdk 8


## Install
 - set JAVA_HOME
 - set M2_HOME and MAVEN_HOME
 - add to path


## Checks

Execute commands :
  
  - java -version

  - mvn -v 


## Run the project

There are two ways


### maven spring runner

Exexute command : 

*mvn spring-boot:run*


### compile and run

Execute command to compile :

*mvn clean package*

Execute the compiled project :

*java -jar target/microscope-spring-basic-0.0.1-SNAPSHOT.jar*

### web

http://localhost:8080/
